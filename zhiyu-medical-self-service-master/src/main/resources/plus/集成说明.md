# 功能代码集成说明

将疾病/药品分页、图片展示、模拟支付、订单功能集成到新项目中。

## 二、数据库要求

### 2.1 订单表（自动创建）

`schema.sql` 会在启动时自动执行 `CREATE TABLE IF NOT EXISTS purchase_order`。

表结构：
```sql
CREATE TABLE `purchase_order` (
  `id`          INT AUTO_INCREMENT PRIMARY KEY,
  `order_no`    VARCHAR(64) NOT NULL UNIQUE,   -- 订单号
  `user_id`     INT NOT NULL,                   -- 用户ID
  `medicine_id` INT NOT NULL,                   -- 药品ID
  `medicine_name` VARCHAR(100),                 -- 药品名称快照
  `medicine_img`  VARCHAR(255),                 -- 药品图片快照
  `quantity`    INT NOT NULL DEFAULT 1,         -- 数量
  `price`       DECIMAL(10,2),                  -- 单价
  `total_price` DECIMAL(10,2) NOT NULL,         -- 总价
  `status`      INT NOT NULL DEFAULT 0,         -- 0待支付 1已支付 2已取消
  `pay_type`    VARCHAR(32),                    -- 支付方式
  `pay_time`    DATETIME,                       -- 支付时间
  `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP,
  `update_time` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 2.2 疾病表添加图片字段

```sql
ALTER TABLE `illness` ADD COLUMN `img_path` VARCHAR(255) DEFAULT NULL COMMENT '疾病图片';
```

> `DatabaseMigrationRunner.java` 会在启动时自动执行此操作。

### 2.3 application.yml 配置

```yaml
spring:
  sql:
    init:
      mode: always
      continue-on-error: true
```

## 四、集成步骤

### Step 1: 复制后端代码

```
将 后端代码/world/  下的所有文件
复制到 你的项目/src/main/java/world/
（覆盖同名文件，新增新文件）
```

### Step 2: 复制前端模板

```
将 前端模板/ 下的 6 个 HTML 文件
复制到 你的项目/src/main/resources/templates/
（覆盖同名文件）
```

### Step 3: 复制静态资源

```
将 静态资源/images/  下的所有文件
复制到 你的项目/src/main/resources/static/assets/images/
```

### Step 4: 添加数据库脚本

```
将 schema.sql 的内容追加到你的 schema.sql，或直接复制到项目 resources 目录
```

### Step 5: 添加配置

在 `application.yml` 中添加：

```yaml
spring:
  sql:
    init:
      mode: always
      continue-on-error: true
```

### Step 6: 添加 MyBatis-Plus 扫描

确保你的启动类上有 `@MapperScan`：

```java
@SpringBootApplication
@MapperScan("world.dao")  // 根据你的包名调整
public class YourApplication {
    public static void main(String[] args) {
        SpringApplication.run(YourApplication.class, args);
    }
}
```

## 五、功能说明

### 分页功能

| 页面 | 路由 | 每页条数 |
|------|------|---------|
| 药品搜索 | `findMedicines?nameValue=xxx&page=1` | 5条 |
| 疾病搜索 | `findIllness?kind=1&illnessName=xxx&page=1` | 9条 |

分页后端逻辑位于 `MedicineService.getMedicineList()` 和 `IllnessService.findIllness()`。

### 图片展示

- 疾病详情页：左侧圆形头像 + 右侧大图
- 图片路径存在数据库 `illness.img_path` 字段
- 启动时 `DatabaseMigrationRunner` 自动更新图片路径

### 模拟支付流程

```
用户搜索药品 → 点击药品 → 药品详情
→ 选择数量 → 点击「立即支付」
→ 创建订单 → 跳转支付中心
→ 30分钟倒计时 → 点击「确认支付」
→ 模拟支付（status 0→1）→ 显示成功
→ 可查看订单列表
```

### 订单状态

| status | 含义 |
|--------|------|
| 0 | 待支付 |
| 1 | 已支付 |
| 2 | 已取消 |
